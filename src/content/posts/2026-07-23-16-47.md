# Linux学习-进程基础操作

### 查看进程

```bash
ps -aux
```

### 查找进程

```bash
ps -aux | grep 进程名
```

### 查看进程树

```bash
pstree
```

### 停止或开始进程

```bash
sudo systemctl stop 进程名
sudo systemctl start 进程名
```

### 查看进程状态

```bash
sudo systemctl status 进程名
```

### 阻止/启用自启动

```bash
sudo systemctl disable 进程名
sudo systemctl enable 进程名
```

### 查看进程列表(活跃的)

```bash
sudo systenctl list-units
sudo systenctl list-units -t 进程类别
```

### 查看所有进程(全部)
```bash
sudo systemctl list-unit-files
```
